# Libros & Saber

**Libros & Saber** es un proyecto de practica que consiste en un aplicativo web desarrollado para la gestión de bibliotecas virtuales. Permite registrar, visualizar, actualizar y eliminar libros fácilmente mediante una interfaz amigable y adaptativa.

---

## Características

- CRUD completo de libros (Create, Read, Update, Delete)
- Interfaz responsive (adaptada a móviles y escritorio)
- Datos persistentes en `localStorage` (por ahora con un archivo .json de 5 libros)
- Navegación clara con menú tipo hamburguesa
- HTML, CSS y JavaScript (sin frameworks)

---

## Tecnologías utilizadas

- HTML5
- CSS3 (con diseño responsivo)
- JavaScript (vanilla)
- JSON para datos simulados
- Git + GitHub para control de versiones
